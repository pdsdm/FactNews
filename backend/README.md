# Backend (.venv)

## Setup

```bash
python3 -m venv .venv
source .venv/bin/activate  # Linux/Mac
# .venv\Scripts\activate   # Windows
pip install -r requirements.txt
```

## Run

```bash
source .venv/bin/activate
python app.py
```

API disponible en: http://localhost:8000
Docs: http://localhost:8000/docs
